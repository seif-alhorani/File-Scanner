# File Scanner frontend
this the front end of the file scanner app which built in using React.js it allow the user to:
- upload file and enter URLs to be scaned from virustotal
- Show the results of the scan
- add notes for each indivisual scan
- delete scan record


# Features
- upload the file with SHA-256 hash generation (client side)
- dynamic result rendering 
- communicate with Express.js backend

# How to run
- must have Node.js (v22)
- npm
- go to the dir file-scanner-frontend
- npm install
- npm start the will start at localhost:3000
